Project Overview

SandBox is a modern and responsive personal portfolio website built using HTML, CSS, and JavaScript.
It is designed to showcase a UI/UX Designer & Developer’s skills, services, and latest projects in a clean and professional layout.

This project highlights front-end development skills including responsive design, animations, structured layout, and interactive components.
